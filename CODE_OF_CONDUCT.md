# Code of Conduct

## Our standards

We expect all contributors to:

- Be respectful and professional
- Assume good intent
- Focus on technical merit
- Accept constructive criticism

This is an **ops-focused infrastructure project**, not a social platform.

---

## Unacceptable behavior

- Harassment or personal attacks
- Aggressive or hostile communication
- Publishing private information
- Intentionally breaking builds or data safety

---

## Enforcement

Project maintainers may:
- Reject pull requests
- Close issues
- Block users from participation

Decisions are made to protect:
- stability
- security
- maintainability

---

## Scope

This Code of Conduct applies to:
- GitHub issues
- pull requests
- discussions
